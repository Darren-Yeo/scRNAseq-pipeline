# scRNAseq-pipeline
scRNA-seq analysis (Seurat) of CK vs HS conditions using publicly available data

## Data Source
The dataset used for this workflow was obstained from the NGDC GSA repository: 
https://ngdc.cncb.ac.cn/gsa (PRJCA042819).

Wen, S., K.Wang, W.Liang, et al. 2026. 
“Transcription Profiling of Potato Leaves in Response to Heat Stress at Single-Cell Resolution.” 
Plant Biotechnology Journal1–18. https://doi.org/10.1111/pbi.70546.

## Code Source
Part of this workflow is adapted from the Seurat scRNAseq tutorial:
https://satijalab.org/seurat/articles/integration_introduction

## Note
This is a demonstration project using publically available data to showcase scRNAseq analysis, not to infer novel insights
